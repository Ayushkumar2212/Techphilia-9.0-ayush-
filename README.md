# Insta-AI | The Neural Matrix Social Experience

![Project Status](https://img.shields.io/badge/Status-Ultra--Pro-blueviolet?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-Pure--JS--|--Tailwind--|--Glassmorphism-blue?style=for-the-badge)

Insta-AI is a high-fidelity, decentralized social media simulation built with a futuristic "Neural Matrix" aesthetic. It features an advanced state management system, real-time interactions, and a premium animated user interface.

## 🚀 Core Features

- **Neural Dashboard**: A glassmorphic primary interface with smooth page transitions.
- **Story Rail**: A dynamic story system supporting both image and video uploads with infinite replayability.
- **Neural Reels**: An immersive, full-screen vertical video player with high-frame-rate aesthetics.
- **Communication Hub**: Secure encrypted messaging (DMs) with integrated audio/video call simulations.
- **Discovery Portal**: A sophisticated search and following system connecting you to over 100 neural nodes.
- **Identity Sync**: A customizable profile matrix where you can edit your essence (bio, name, and avatar).
- **Persistent State**: All your actions—like counts, follows, and saved items—are archived in your browser's local memory.

## 🛠️ Technology Stack

- **Core**: [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) & [Vanilla JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for a rapid, high-performance UI.
- **Icons**: [Lucide Icons](https://lucide.dev/) for crisp, futuristic iconography.
- **Typography**: [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans) via Google Fonts.
- **Animations**: [Canvas API](https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API) for neural background simulations and native CSS transitions.
- **Data Persistence**: [localStorage API](https://developer.mozilla.org/en-US/docs/Web/API/Window/localStorage) for client-side state management.
- **Backend Simulation**: [Fetch API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API) with native JSON synchronization.
- **Local Server**: [PowerShell HTTP Listener](https://learn.microsoft.com/en-us/dotnet/api/system.net.httplistener) for zero-dependency hosting.

## 🛠️ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, or Edge).
- (Optional) PowerShell for local hosting.

### Local Hosting
To experience the portal with full functionality (including cross-origin assets), it is recommended to run the local server:
1. Open PowerShell in the `Tech/` directory.
2. Run the command: `powershell -ExecutionPolicy Bypass -File serve.ps1`.
3. Visit the URL: `http://localhost:8081`.
