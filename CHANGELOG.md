# Changelog

All notable changes to the Insta-AI Matrix will be documented in this file.

## [v3.5.0] - 2026-04-20
### Added
- **Global Profile Viewing**: Every user in the matrix now has a unique, interactive portal.
- **Node Discovery Grid**: A new 30-tile explore interface with high-end hover effects.
- **Relay System**: Full implementation of Like, Comment, and Share (Relay) across all feed nodes.
- **Archive System**: Persistent "Saved" tab in the profile matrix for bookmarked content.
- **Neural Comms**: Simulation of Secure Audio and Video Calls within the DM center.
- **Data Persistence**: State is now synchronized with the local neural storage (localStorage).

## [v2.1.0] - 2026-04-18
### Added
- **Reels Engine**: Full-screen vertical video player with sound support.
- **Story Rail**: Initial implementation of the infinite viewing story rail.
- **Identity Sync**: First version of the Edit Profile and Settings module.

## [v1.0.0] - 2026-04-12
### Added
- **Initial Matrix**: Basic structural setup for Login, Signup, and Feed.
- **Neural UI**: Core glassmorphism and animated canvas background.
- **Lucide Integration**: Standardization of icons across the platform.
